What this page covers Adding items, entering progress, moving across dates, reviewing history, changing settings, premium features, and backup.

1. Main structure

HabiDone uses four item types.

  • Habit: things you want to keep doing
  • Task: items with a due date or target amount
  • Limit: actions you want to reduce or cap
  • Diary: daily notes and records

All of them appear in one list so you can record and review daily activity without switching screens.

2. Getting started

Add items

Use the + button in the top right of the main list to add a new Habit, Task, Limit, or Diary entry.

Enter progress

You can enter progress directly from the list.

  • Tap: add progress
  • Long-press: enter a value directly

Diary items use text entries, while habits, tasks, and limits use numeric progress or counts.

Move to other dates

Use the date at the top of the list and the left/right buttons to move between past and future dates.

  • Past dates: review or add past records
  • Future dates: check upcoming items and due-date visibility

You can always return to today by scrolling the list upward.

3. Reading the list

Guide display

Each item can show guide text such as remaining days, required pace, count progress, or deadline information. The guide style can be changed in Settings.

  • Remaining + Pace
  • Remaining + Count
  • Remaining + Progress
  • Deadline + Pace
  • Deadline + Count
  • Deadline + Progress
  • Hidden

Status labels

Items may show states such as Active, Ready, Must Do, Do Not, or Over. These are calculated automatically from the item type, schedule, and current progress.

Edit or review an item

Use the … button on each item to edit it, duplicate it, open the monthly summary, or review history. For Diary items, you can open All Diaries.

4. How item settings work

Schedule

You can combine start dates, end dates, recurrence, weekdays, month dates, months, and flexible intervals based on the last completion date.

Show Early

Tasks can be shown a few days before their due dates.

  • Show Early: how many days in advance to show the item

Some Habit setups use Active Before Due to define how many days before a due date the item becomes active.

Secret

When Secret is enabled, premium users can mask the item title and diary text with passcode protection.

Color Tag

Color tags help categorize entries. For Diary items, you can also set the title from the selected tag name.

5. History and review

History

Habits, tasks, and limits support daily history review. Diary entries can be opened through All Diaries.

Diary history shows not only the title but also up to three lines of body text.

Monthly summary

Monthly summaries help you review state distribution and trends for each item. You can move across months and inspect the breakdown in more detail.

6. What you can change in Settings

Filter / Ordering

  • Period filter
  • Minimum priority filter
  • List ordering: Custom / Color Tag / Importance / Nearest Due / Execution Priority
  • Item type ordering

Notifications

There are two notification types.

  • Check-in Notification: sent near the end of your day to remind you to record progress
  • Plan Notification: sent at the start of your day to review what is due

Notifications can summarize how many items need attention today, along with other items you may want to review.

If notifications are disabled at the system level, you can open notification settings from the app.

Day Boundary

Start of Day Offset lets you shift the in-app start of the day, which is useful if your daily routine crosses midnight.

Theme

The Theme screen lets you change:

  • Appearance: System / Light / Dark
  • Layout: Accent / Card
  • Guide style on the list
  • Color presets and custom colors
  • Color intensity

Backup

Premium users can use Create Backup and Restore Backup to export or restore items, records, and tags.

7. Premium features

The core features are free to use. Premium is a one-time purchase that unlocks the additional features below.

Premium includes:

  • Ad-free experience
  • Custom colors and saved presets
  • Masked display for Secret items
  • Backup creation and restore

You can review the details, purchase, or restore from Check Premium Features in Settings.

8. Common usage patterns

Track habits

Create a Habit and set a daily or weekly recurrence. You can then update it directly from the list.

Manage deadline-based work

Create a Task, set an end date, and use Show Early if you want the task to appear before its due date.

Reduce unwanted actions

Create a Limit item to track the maximum amount or count you want to stay under within a period.

Keep notes or a daily journal

Create a Diary item to store personal notes. You can later review them from All Diaries.
